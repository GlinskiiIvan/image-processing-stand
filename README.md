1) Создание окружения, если не создано

   Linux:
```shell
python3 -m venv venv
```

   Windows:
```shell
python -m venv venv
```

2) Активация окружения 

   Linux:
```shell
source venv/bin/activate
```
  
  Windows:
```shell
.\venv\Scripts\Activate
```

3) Установка зависимостей

   linux:
```shell
pip install requirements.txt    
```
  
  Windows:
```shell
python -m pip install -r requirements.txt
```

4) Запуск скрипта

   Linux:
```shell
python3 name.py
```
  
  Windows:
```shell
python name.py
```
